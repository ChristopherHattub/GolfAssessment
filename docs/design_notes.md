# Design Notes
Treated this timed assignment as a basic implementation of the requirements, keeping it lightweight and efficient.

1. Utilized using an integer for timeslot so that would handle input error at db level.


2. Kept project simple to eliminate framework depencies and highlight core programming.

3. Modular seperation of code  and helper functions for max readibility.

With More time i might add in user authentification services to protect the integrity of user reservations. Also would probably switch to an async framework to support concurrent requests. Switching to a modern framework would lighten things up. There are also some magic number that could be defined, and helper function modularization that could happen.
